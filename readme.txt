eBay Shop page on your WordPress website or blog.

== Description ==

= Do you need a solution to list your eBay shop products on your WordPress blog or website? =

This plugin allows you to add a products page on your WordPress website to list products form a selected eBay seller or shop. You can also add a search form in the sidebar that allows website visitors to search by brand,  category, size, or filter by price range.

You will need an eBay App ID, which can be obtained by signing up for a developer account here, 
[http://go.developer.ebay.com/developers/ebay](http://go.developer.ebay.com/developers/ebay "eBay API Key")


= Major features in eBay Shop Page include: =

* Automatically list products from eBay Seller or Shop into your website or blog.
* Contains Sidebar widget for Search filter.
* Search by Category, Brand, or filter by Price Range.

Please Note: You'll need an [ebay.com App ID](http://go.developer.ebay.com/developers/ebay) to use it. eBay App IDs are free for personal websites and blogs.  

== Installation ==

Upload the eBay Shop Page plugin to your blog, Activate it, then enter your [ebay.com App ID](http://go.developer.ebay.com/developers/ebay).

Go to the "eBay Shop Page" admin page, enter your App ID, Seller name, and Select a page to display the Shop.

1, 2, 3: You're done!

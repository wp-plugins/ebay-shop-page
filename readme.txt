== Description ==

Allows you to add a Shop page on your WordPress website to list products form a selected eBay seller or shop. You will need an eBay App ID, which can be obtained by signing up for a developer account, here, http://go.developer.ebay.com/developers/ebay


Major features in eBay Shop Page include:

* Automatically list products from eBay Seller or Shop into your website or blog.
* Search by Category, Brand, or filter by Price Range.
* Contains Sidebar widget for Search filter.

Please Note: You'll need an [ebay.com App ID](http://go.developer.ebay.com/developers/ebay) to use it.  App ID are free for personal websites and blogs.  

== Installation ==

Upload the eBay Shop Page plugin to your blog, Activate it, then enter your [ebay.com App ID](http://go.developer.ebay.com/developers/ebay).

Go to the "eBay Shop Page" admin page, enter your App ID, Seller name, and Select a page to display the Shop.

1, 2, 3: You're done!
